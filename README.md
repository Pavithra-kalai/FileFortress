# 🔐 Secure File Sharing System

## 📌 Description
A web application that allows users to securely upload, store, and share files. Files are encrypted before storage to ensure data security and privacy.

---

## 🚀 Features
- User Registration & Login
- Secure File Upload
- File Encryption & Decryption
- File Download
- Shareable File Links

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: SQLite
- Security: Cryptography (Encryption)

---

## 📂 Project Structure
secure-file-sharing-system/
│── backend/
│── frontend/
│── database/
│── README.md


---

## ▶️ How to Run

### 1. Backend Setup
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py


### 2. Frontend
- Open `frontend/index.html` in browser
